<h4>1. Preparing your drawing for animation</h4>

<p>The first step for this is preparing your drawing/character/logo etc... for animation. In this case I will use this illustration so you can see the structure you need it, if you are using another drawing program instead of <b>Adobe Photoshop</b>, you have to export it to <b>PSD, </b>so <b>After Effects</b> can read all the layers. Also, if parts are in individual folders, is better to save as a new <b>PSD</b> version with all layers rasterized, there are many ways to import to <b>After Effects</b>, but I will show here the easier one for me.</p>

<p>Also, If you don´t want to use a <b>PSD</b> file, you can also export each part separately in <b>PNG</b> and arrange them manually, but its a little more tedious.</p>

<h4> 2. Importing our character to After Effects</h4>
<p>Now you can close your character in <b>PSD</b> and open <b>After Effects</b>, and when its open, just create a new project. Then we go to File Import and select our <b>PSD</b> File. For this Tutorial we are gonna select <b>Composition → Preserve layers size</b>, also select Editable Layers.</p><p >When the file is imported correctly, you will see that <b>After Effects </b> created one folder with all layers and a new composition. If you double click on it you will see our composition open (don't pay attention at the fact that my software is in Spanish, tools are the same).</p>

<div class='mb-3 text-center'><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEh8pzyke85N2IahPanEFkeDIuzxqOnakmMGuSAbr0XtAxDfYtTn7ro-Cn3JcFs7hF7tpQs0lkEz4HoC1FLZOR7QJW8zYlmzj26u2GZFw3Pf2ySWT2lAuqp2DPbdXZaaPYLz0mJCRM1a_pNbdv-VAsiVwIPPG0ehft-ENeo1hEHEjxQlRBg22Zxew2TITA/s1600/AfterEffects_02.jpg"><img alt="After Effects 02" border="0" data-original-height="995" data-original-width="1092" src="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEh8pzyke85N2IahPanEFkeDIuzxqOnakmMGuSAbr0XtAxDfYtTn7ro-Cn3JcFs7hF7tpQs0lkEz4HoC1FLZOR7QJW8zYlmzj26u2GZFw3Pf2ySWT2lAuqp2DPbdXZaaPYLz0mJCRM1a_pNbdv-VAsiVwIPPG0ehft-ENeo1hEHEjxQlRBg22Zxew2TITA/s1600/AfterEffects_02.jpg"/></a></div>

<h4>3. Preparing Layers for Animation</h4>
<p>Now is time to link our body parts. At the right of the Spiral icon, we just select the body part where each layer will be attached to. So when you move or rotate, elements will move with it. If you have rigged a character in the past, this will see you very familiar, in this character we are gonna use the hip as main layer. You can also link layers with the spiral icon, just dragging the spiral to the attached body part, its your choice.</p>

<div class='mb-3 text-center'><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEgyMf1J-y3UBO0jXzeDC1um4AfU2bfpxhexNT-U8z7QRAucfm_-pvIEv3mcKjK6WXKPCbYy47Yf37OWU7Y_KiY3WgtCok3rvV-K2BClQ4AEfmJpznR6tcNRUfTwVRBYi8lW1KagboxyM_0Kswvg0OCw_uO4QH3l8k-i7AiHNB7odmC1Qq4sYviRolFV7w/s1600/AfterEffects_03.jpg"><img alt="After Effects 03" border="0" data-original-height="995" data-original-width="1092" src="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEgyMf1J-y3UBO0jXzeDC1um4AfU2bfpxhexNT-U8z7QRAucfm_-pvIEv3mcKjK6WXKPCbYy47Yf37OWU7Y_KiY3WgtCok3rvV-K2BClQ4AEfmJpznR6tcNRUfTwVRBYi8lW1KagboxyM_0Kswvg0OCw_uO4QH3l8k-i7AiHNB7odmC1Qq4sYviRolFV7w/s1600/AfterEffects_03.jpg"/></a></div>
    
<p>You can test if all links work well selecting the main layer (in this case the hip) and trying to move or rotate it. If it works well, all character will be moving with it, if this is ok, you can continue to fix pivot points and add puppets points.</p>

<h4>4. Puppet points VS Pivot Point movements</h4>

<p>Ok, now we are gonna fix our layers one by one so they will be ready for animation. This is the last step for rigging our character. For selecting each one of the layers we just have to double click on the layer name we want to modify.</p>
<p>For default we have a black background for our file, and in some cases (as the one in this tutorial) our characters contains a lot of black and we need another color for background. That's very easy to change, you just need to enter in <b>Composition → Composition settings</b>, and at the bottom you just select background color to one is better for you.</p>

<div class='mb-3 text-center'><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEhlFNBDpaHljU3VoifCN9HhurfVFC0dyv84VBlnSICOAq6sw29B83_qOM_qMXBsPwybO95EqQ7kSbiVdMo6ownoEtqLLbVawS4AqrDV_WSGphgsD4Ham_2y9s9PC_Ednx8j_pEv4gnikgQIIbmUF1sU_sO8egcAcPJUaIqVFOdPxYdyRFl0iF_fBnwJsQ/s1600/AfterEffects_04.jpg"><img alt="After Effects 04" border="0" data-original-height="995" data-original-width="1092" src="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEhlFNBDpaHljU3VoifCN9HhurfVFC0dyv84VBlnSICOAq6sw29B83_qOM_qMXBsPwybO95EqQ7kSbiVdMo6ownoEtqLLbVawS4AqrDV_WSGphgsD4Ham_2y9s9PC_Ednx8j_pEv4gnikgQIIbmUF1sU_sO8egcAcPJUaIqVFOdPxYdyRFl0iF_fBnwJsQ/s1600/AfterEffects_04.jpg"/></a></div>

<p>Now we have to select the pan tool so when we moved our pivot points our character don't move from their original position. In this case, the wing come from the back so the point it stick the wing to the body is the perfect point for our pivot. This wing will not have puppet points because we just need to rotate them, so we just fix the pivot point in this layer.</p>

<div class='mb-3 text-center'><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEjOsfHrNSiLg0bGhk8fqrpKDNugumXhICtEQ4_Wqj_PFonJ13eK0913SWrWX6lFipInHNWXoIlD-YlhqeD2oiYX7TnvMCZ8RTniYDOd_TVZOPeToThgGtGe2qK0UH2INBz8U0s-2iNxAQNhiLGN7PIeNUOy06IOuh8la5-10UdYUNE3kL0niZSYNR26Jg/s1600/AfterEffects_05.jpg"><img alt="After Effects 05" border="0" data-original-height="995" data-original-width="1092" src="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEjOsfHrNSiLg0bGhk8fqrpKDNugumXhICtEQ4_Wqj_PFonJ13eK0913SWrWX6lFipInHNWXoIlD-YlhqeD2oiYX7TnvMCZ8RTniYDOd_TVZOPeToThgGtGe2qK0UH2INBz8U0s-2iNxAQNhiLGN7PIeNUOy06IOuh8la5-10UdYUNE3kL0niZSYNR26Jg/s1600/AfterEffects_05.jpg"/></a></div>

<p>We just need to adjust pivot points to layers we just need to rotate or move. In other hand, the puppet points will deform our drawing and in some cases they help to give more life and flexibility to our character. For example, in my case I will just add puppet points to the hair, mouth and antennae's. Sometimes we get exited with this new tool, but you will see you will not need this for every movement. Anyway, you can add more puppet points later if you want to give more movement in the future.</p>
<p>For adding puppets points you just need to select the <b>puppet selection tool</b> and start adding <b>puppet points</b>, yes, just like that. You can also test them in the layer preview, so you can see if the movement in correctly for this body part. Test and add as many puppet points as you need, but not too many, because animating can became a mess in our timeline. When this is done, we are ready to start animating.</p>

<div class='mb-3 text-center'><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEhN22a8ny_p6qOqzvQ-YY5oH2w7XqkO6-Ut_mIuLWrKtfM_hg7a24LQlktZtZyf9ywYrCqf3K5-KA0-liQAeYxjITc3o5GPjiEyS-6EdYNq5VGp1urBB9thuWpzIO7hSmfEDTAvf083M3TiwhMfEPFI_A3mzEBpmvBlstLdb5CUrf2CcH4lPixJ9j3zNA/s1600/AfterEffects_06.jpg"><img alt="After Effects 06" border="0" data-original-height="995" data-original-width="1092" src="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEhN22a8ny_p6qOqzvQ-YY5oH2w7XqkO6-Ut_mIuLWrKtfM_hg7a24LQlktZtZyf9ywYrCqf3K5-KA0-liQAeYxjITc3o5GPjiEyS-6EdYNq5VGp1urBB9thuWpzIO7hSmfEDTAvf083M3TiwhMfEPFI_A3mzEBpmvBlstLdb5CUrf2CcH4lPixJ9j3zNA/s1600/AfterEffects_06.jpg"/></a></div>

<h4>5. Animating our character</h4>

<p>If we want to make a loop animation, we just need to make one time loop. After I will tell you how to maintain the looping as long as you want. But for now, we are just gonna make our butterfly flapping her wings. So we are gonna set our time for this movement. Now we go back to timeline settings and decide the duration for this loop, in my case I want one second duration.</p>

<div class='mb-3'><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEj1-VI4eoFkAKZczUfo-c5eYeCKAxBUQeIaZlaz32NUC-Z7Ht2mn9kNo1z1lmd5NYqBFhv9Iha_07_L_sUvcZxiozRpv-b6kbvbkyGB733g5vHKIUkyld2S45Q5V1ojqy7GaH_hlSoCwXNIanMQDjmEeWRRdZ81ALz15hz-fBqFNhkIPvnwFDigdG4_qA/s1600/AfterEffects_07.jpg"><img alt="After Effects 07" border="0" data-original-height="995" data-original-width="1092" src="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEj1-VI4eoFkAKZczUfo-c5eYeCKAxBUQeIaZlaz32NUC-Z7Ht2mn9kNo1z1lmd5NYqBFhv9Iha_07_L_sUvcZxiozRpv-b6kbvbkyGB733g5vHKIUkyld2S45Q5V1ojqy7GaH_hlSoCwXNIanMQDjmEeWRRdZ81ALz15hz-fBqFNhkIPvnwFDigdG4_qA/s1600/AfterEffects_07.jpg"/></a></div>

<p>If you have animated before, you will just know that animation came first from outside movements and then we make inside movements. In my case, we are gonna animate the wings and body and then we will add all the details, such as head, hair, and legs.</p>
<p>For wings and body we are just gonna use Rotation and Position movements. Keyboard Shortcuts are <b>P </b>for position, and <b>R</b> for rotation. So now we select our main layer (hip layer) and just press <b>P</b> with the keyboard so we can change the position of our character.</p>

<div class='mb-3 text-center'><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEinDvFFd5psTXxG6Tbw0Jem7kk6Q8X7iWVUyli5LVX56PpG2ikbVmpjCHbEdaZjc4K-AJ2ZRaxMvsn6uF3d6GLmBP0cLRil-QKPl3ytNik4Yg4qQOMYQfHfNyZr7Fk_AfIsH9O8ohtd09LBcIEPNSKwGqSZevFpIlJTxAIcsZXquVXW-THgWG8PlLnDGw/s1600/AfterEffects_08.jpg"><img alt="After Effects 08" border="0" data-original-height="995" data-original-width="1092" src="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEinDvFFd5psTXxG6Tbw0Jem7kk6Q8X7iWVUyli5LVX56PpG2ikbVmpjCHbEdaZjc4K-AJ2ZRaxMvsn6uF3d6GLmBP0cLRil-QKPl3ytNik4Yg4qQOMYQfHfNyZr7Fk_AfIsH9O8ohtd09LBcIEPNSKwGqSZevFpIlJTxAIcsZXquVXW-THgWG8PlLnDGw/s1600/AfterEffects_08.jpg"/></a></div>

<p>In my case as it will be a loop we will need that the animation begin and end in same position, so before animating we settle first position and then select the keyframe and press <b>Ctrl+C </b>and go to the last frame position and paste your frame with<b> Ctrl+V</b>. Now we make the movements we want in the middle of them, in my case, I will just add one more keyframe in the middle.</p>
<p>Now for wings, we just need to press R in keyboard so we can just rotate them. Also select rotation tool in the tool panel. I will do the movement in the same keyframe so our boucle works fine with wings movement, but you can animate your loop as you wish. You can continue with all simple movements with this tools.</p>
  
<div class='mb-3 text-center'><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEi95lc6nB1fVHiwnr0-Q9MmR5dzxWhL-oH6C-LznT7y6Ch9ztvnK76qB2I70opZKQ6dBnr2CDvzPNeE5xFK87IEoiZ7geNemvg3CGXvSrnF-9qLreEvWIrodKLfLGoO88z4AvEzbj1CjKQgfUL7B_qgmfiwIVkzWHFLcqXSNjkaJ7V-tVjQRMpMSlGh5A/s1600/AfterEffects_09.jpg"><img alt="After Effects 09" border="0" data-original-height="995" data-original-width="1092" src="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEi95lc6nB1fVHiwnr0-Q9MmR5dzxWhL-oH6C-LznT7y6Ch9ztvnK76qB2I70opZKQ6dBnr2CDvzPNeE5xFK87IEoiZ7geNemvg3CGXvSrnF-9qLreEvWIrodKLfLGoO88z4AvEzbj1CjKQgfUL7B_qgmfiwIVkzWHFLcqXSNjkaJ7V-tVjQRMpMSlGh5A/s1600/AfterEffects_09.jpg"/></a></div>

<p>Now we are gonna detail our animation. You will see that your timeline will be a little messy in the next steps. But no worries, you can use the shy tool to clean up a little. When shy tool is activated all layers selected as shy layers will be hidden in our timeline, so its just switch on an off for having control in our timeline.</p>

<div class='mb-3 text-center'><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEg5G9m41IgKoVFDIVm3py3GcrL36Db_U5cDbIc2qLko-ayvkaRSm96yxzIqu2GZGI7Dh9JsVyGW6w4r1XT2LmPnuhcX41An-rFBzT9PfSRU8wGQ0-kSDxFMMFdmuurItcfSLDgH2ixIrQazOR4Jn0SvOlDJ5ycJftHvQbfbkyIQjeY7PQnPx_84Ynbrgg/s1600/AfterEffects_10.jpg"><img alt="After Effects 10" border="0" data-original-height="995" data-original-width="1092" src="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEg5G9m41IgKoVFDIVm3py3GcrL36Db_U5cDbIc2qLko-ayvkaRSm96yxzIqu2GZGI7Dh9JsVyGW6w4r1XT2LmPnuhcX41An-rFBzT9PfSRU8wGQ0-kSDxFMMFdmuurItcfSLDgH2ixIrQazOR4Jn0SvOlDJ5ycJftHvQbfbkyIQjeY7PQnPx_84Ynbrgg/s1600/AfterEffects_10.jpg"/></a></div>

<p>Now we are gonna animate the hair, so we need to animate all the puppet points we have. For this we will select the layer we want to modify and then we are gonna press <b>U</b> in keyboard, that will open all puppet layers. I recommend animate layer one by one, so it won't be too messy.</p>

<div class='mb-3 text-center'><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEi4s3Uq0hElzsZ1Lwj6k88JwlFu-HmVp3avMrYOJ0zI3_igmN2OmcH0ZnoHMqPuBEOcCEBu2x11dq8NV528xfDAin5UtaeJB9jD7BtsCiGtGoZPEYniIflC5-VoiKcRZ_O0oE5e9AtT4o8eKOW5jZUm8hG1SajC4KfXj6caUhj0eiRnDSE5Mwqi5j_QSA/s1600/AfterEffects_11.jpg"><img alt="After Effects 11" border="0" data-original-height="995" data-original-width="1092" src="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEi4s3Uq0hElzsZ1Lwj6k88JwlFu-HmVp3avMrYOJ0zI3_igmN2OmcH0ZnoHMqPuBEOcCEBu2x11dq8NV528xfDAin5UtaeJB9jD7BtsCiGtGoZPEYniIflC5-VoiKcRZ_O0oE5e9AtT4o8eKOW5jZUm8hG1SajC4KfXj6caUhj0eiRnDSE5Mwqi5j_QSA/s1600/AfterEffects_11.jpg"/></a></div>
  
<p>Now is time to set up all details for our loop animation. And when we finished we can make it longer as we need need and also add some effects.</p>

<h4>6. Making loops and adding more details to our animation</h4>

<p>Ok, now we are gonna make a new composition so we can manage our animation better. So just click on<b> Composition</b><b> → </b><b> New Composition</b>. This new composition will have same configuration as last one, but now I want that the composition will be ready for YouTube, so I will select the <b>HDV 1080 25</b> Preset and give it a duration of 10 seconds, but if you want another settings feel free to test them.</p>
  
<div class='mb-3 text-center'><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEjPhf3Qo3gcSc08p67jjCo3_Nix31FahR5Qqna-WK0dv1B45--8l9RGKG6On1yPlOM7J4yDufpwuO4_qafoZ54BJf4nnClU5dq32DXhWstZcXjmC2RFbjiGQkXdrU7EjDSnTYU7RDjnZY3yaPyNEKo21Otl5IreuQSVMaAwylgkUwvc5wK-6oYxEn9k0A/s1600/AfterEffects_12.jpg"><img alt="After Effects 12" border="0" data-original-height="995" data-original-width="1092" src="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEjPhf3Qo3gcSc08p67jjCo3_Nix31FahR5Qqna-WK0dv1B45--8l9RGKG6On1yPlOM7J4yDufpwuO4_qafoZ54BJf4nnClU5dq32DXhWstZcXjmC2RFbjiGQkXdrU7EjDSnTYU7RDjnZY3yaPyNEKo21Otl5IreuQSVMaAwylgkUwvc5wK-6oYxEn9k0A/s1600/AfterEffects_12.jpg"/></a></div>

<p>Now I will add a background to my animation. In this case I will use a solid background. So I right click in my timeline and select New Solid. That will add a solid color to our background. I want to be a gradient background so I add an effect for that. I will add it by selecting Effect Generate Ramp (Spanish will be <i><b>Gradación de Degradado</b></i>), there we select our colors and can play with the effect options.</p>

<div class='mb-3 text-center'><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEh0nVcJa0DSky3XqYbP4bAlp6PJHMX59aLpfSSc1zMNOAscJzHP2X_owafbWcBBwoGtCFBj3VhltQz2jBuElwG78CbjCfYJz4TJM8kR2gR2yDFIvDMzenMV7XqGWHsvpXBQaknybI4iLrFSq-Ngr0vNLbQyHllc49jUMbtgqyKi9qTW4T7idyoHtv-Cdw/s1600/AfterEffects_13.jpg"><img alt="After Effects 13" border="0" data-original-height="995" data-original-width="1092" src="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEh0nVcJa0DSky3XqYbP4bAlp6PJHMX59aLpfSSc1zMNOAscJzHP2X_owafbWcBBwoGtCFBj3VhltQz2jBuElwG78CbjCfYJz4TJM8kR2gR2yDFIvDMzenMV7XqGWHsvpXBQaknybI4iLrFSq-Ngr0vNLbQyHllc49jUMbtgqyKi9qTW4T7idyoHtv-Cdw/s1600/AfterEffects_13.jpg"/></a></div>

<p>Ok, now the loop part is a little complicated, but once you have done it for the first time, next ones will be easier. For the animation became looping all the time we have to go to <b>Layer → Time → Enable Time Remapping</b>, that will create two keyframes at the beginning and at the end, now you just have to create a new keyframe before last one. Now on the last keyframe we are gonna change the time to zero.</p>

<div class='mb-3 text-center'><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEjCv5wabL2Ecuu1lju79lIVLerlKGsL92Bodp2n8GcLf5jaXoLdr7ydfZhSD3IhGS_QrquJg1uFnt8zXxqC4Ko9-wifbBBbZEdUKHFjxAQXgkvywg1y5UYK80JuMedcksHTus5X07ATdvwLYXkcQ_m8nww8ZvhEhsuTbL0rPg4KmylFYV9Wo_CUy99J0w/s1600/AfterEffects_14.jpg"><img alt="After Effects 14" border="0" data-original-height="995" data-original-width="1092" src="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEjCv5wabL2Ecuu1lju79lIVLerlKGsL92Bodp2n8GcLf5jaXoLdr7ydfZhSD3IhGS_QrquJg1uFnt8zXxqC4Ko9-wifbBBbZEdUKHFjxAQXgkvywg1y5UYK80JuMedcksHTus5X07ATdvwLYXkcQ_m8nww8ZvhEhsuTbL0rPg4KmylFYV9Wo_CUy99J0w/s1600/AfterEffects_14.jpg"/></a></div>

<p>Now we delete our last keyframe. Now we are gonna select the Time remapping layer and open in main menu <b>Animation → Add Expression</b>. You will see that can add expressions now. Click on the menu Expression language and open and select<b> Property → LoopOut(Type="Cycle", numKeyframes = 0)</b>, and now it will be shown in the text box. Now is ready, you just have to make your layer longer, in this case, as long as the composition.</p>
